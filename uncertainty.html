Custom R Package Built to Asses Uncertainty in Datasets (Logistic Regression)

The R package developed here offers a comprehensive suite of tools for evaluating binary classification models, particularly logistic regression models. At its core is the MakePredictions function, which ensures model compatibility, checks input validity, and generates probabilistic predictions alongside observed outcomes. Building on this foundation, the package introduces several analytic functions such as calculate_EPE, set_cost, and calculate_misclassification_cost, which allow users to assess model performance not just through accuracy but also through cost-sensitive metrics that reflect real-world decision impacts. Additionally, it provides key tools to explore classification trade-offs through metrics like TPR, FPR, precision, and optimal thresholding using ROC curve analysis.

Visualization is another strength of the package, with built-in functions like generate_roc and confusion_matrix_metrics_plot to graphically assess model performance. These visualizations, coupled with explanatory output, help users better understand the behavior of their classifiers and make informed decisions about threshold selection and model refinement. The package emphasizes interpretability and flexibility, making it particularly useful for data scientists or analysts working in domains where misclassification costs and prediction reliability are critical.
