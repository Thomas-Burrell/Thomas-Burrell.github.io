API and ETL Project for Analyzing TV Show Data

This project showcases an end-to-end ETL pipeline and data analysis workflow built using Python to explore trends in popular U.S. television shows. The data is sourced from the TVMaze REST API, which provides comprehensive metadata on top-rated TV shows.

Using Python's requests library, the API is queried to retrieve raw JSON data, which is then parsed and transformed into structured formats for analysis. The pipeline extracts key variables such as runtime, genre, language, and more, offering insights into what defines today's most-watched TV shows.
